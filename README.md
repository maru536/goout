/android/ 					>>> android app source code
/apk/ 						>>> android app apk
arduino/goout_release_v1.0 	>>> arduino source code
arduino/libraries			>>> arduino libraries code

========= how to execute ===========
1. apk를 안드로이드 기기에 설치
2. arduino/libraries 안의 폴더들을 아두이노 설치 폴더 내 libraries 폴더로 복사
3. arduino IDE > 스케치 > 라이브러리 포함하기 > 라이브러리 관리 > VL53L0X 검색 > VL53L0X by Pololu 1.0.2버전 설치
4. arduino/goout_release_v1.0를 보드에 심는다.
